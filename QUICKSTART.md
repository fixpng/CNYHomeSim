# 快速开始指南

## 安装和运行

1. **安装依赖**
```bash
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

3. **访问游戏**
打开浏览器访问 `http://localhost:3000`

## 游戏流程

### 第一步：开局设定
- 选择**自定义模式**或**随机模式**
- 自定义模式：手动选择工作单位、职业、薪资等
- 随机模式：一键生成所有属性

### 第二步：返乡之路
- 根据距离和余额选择交通方式
- 注意：高铁需要抢票，有失败风险！

### 第三步：9天春节日常
- 每天有3点体力（早/中/晚各1点）
- 选择不同的行动会影响健康、精神、余额、口碑
- 部分事件有子选项，需要进一步选择

### 第四步：返程总结
- 查看属性变化曲线
- 获得最终评价（S/A/B/C级）
- 查看趣味统计

## 游戏技巧

1. **合理分配体力**：每天只有3点体力，要合理选择
2. **注意余额**：买年货、发红包都会花钱，要量力而行
3. **关注属性**：健康太低会生病，精神太低会emo
4. **家庭关系很重要**：家庭关系好，很多互动会有正面效果
5. **随机事件**：第3-7天会有随机事件，增加游戏趣味性

## 常见问题

**Q: 游戏进度会保存吗？**
A: 是的，游戏会自动保存到localStorage，刷新页面不会丢失进度。

**Q: 可以重新开始吗？**
A: 在总结页面点击"再来一次"按钮即可重新开始。

**Q: 手机端可以玩吗？**
A: 可以，游戏完全适配手机端，响应式设计。

## 开发说明

- 技术栈：Vue 3 + Vite + ECharts
- 状态管理：使用Vue的响应式系统
- 数据持久化：localStorage
- 构建：`npm run build` 生成dist目录
