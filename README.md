# 打工人春节回家模拟器

一款适配电脑和手机的纯前端网页版"打工人过年回家模拟器"，核心围绕9天年假设计互动玩法，通过初始/随机属性影响返乡方式、每日互动选择和最终结局。
当前版本已做结构化整理与稳定性修复，支持直接本地运行体验。

## 游戏特色

- 🎮 **9天春节假期体验**：从返乡到返程，完整模拟春节假期生活
- 🎲 **丰富的属性系统**：工作单位、职业、薪资、家庭关系等多维度属性
- 🚄 **真实的返乡体验**：根据距离和余额选择交通方式，体验抢票的紧张
- 💰 **经济系统**：余额管理，花钱买年货、发红包，收红包增加收入
- 📊 **属性变化追踪**：健康、精神、口碑、余额实时变化，影响游戏结局
- 📱 **手机端适配**：响应式设计，触屏友好，按钮≥44px

## 技术栈

- Vue 3 + Vite
- ECharts（用于属性曲线图）
- localStorage（进度保存）

## 游戏玩法

### 1. 开局设定

- **自定义模式**：手动选择工作单位、职业、薪资等属性
- **随机模式**：一键生成随机属性，更有挑战性

### 2. 返乡之路

根据距离和余额选择交通方式：
- 同城：公交/地铁
- 邻市/省内：大巴/高铁（抢票）
- 跨省：普通火车/高铁（抢票）
- 远跨省：绿皮火车/飞机

### 3. 9天春节日常

每天有3点体力，可以选择不同的行动：
- **第1天（除夕）**：帮父母打扫、买年货、看春晚
- **第2天（大年初一）**：拜年、接待亲戚、家庭聚餐
- **第3-7天**：走亲访友、随机事件（同学约酒、亲戚借钱、被介绍相亲等）
- **第8天（初七）**：返程准备、抢返程票

### 4. 返程总结

- 属性变化曲线图
- 最终评价（S/A/B/C级）
- 趣味统计（总花费、被催婚次数、喝酒次数等）

## 属性系统

### 初始属性
- 工作单位：私企、外企、国企、事业单位等
- 职业：程序员、销售、教师、外卖员等
- 薪资范围：2k以下到50k+
- 工作地/老家：超一线到六线（农村）

### 随机属性
- 家庭关系：仇视 → 相亲相爱
- 原生家庭经济：一贫如洗 → 富可敌国
- 婚姻状态：单身/恋爱/已婚/离异
- 年龄：20-50岁
- 技能：会喝酒/社牛/社恐/会做饭
- 负债：无/小额/大额

### 动态属性
- **健康**（0-100）：熬夜/喝酒/赶路降低，休息提升
- **精神**（0-100）：被催婚/攀比降低，聊天愉快提升
- **余额**：花钱减少，收红包增加
- **口碑**（0-100）：正向互动提升，反向互动降低
